## 其他
1 关于SCPSPacketSize的说明
        SCPSPacketSize是机器视觉摄像机向客户端发送数据包的大小。
        SCPSPacketSize默认值为8164.
        网卡未开启巨帧或网卡巨帧设置过小或交换机不支持巨帧，会导致摄像机剧烈掉帧丢包。
        SCPSPacketSize设置值大于网卡巨帧设置值会导致图像花屏。
